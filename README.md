
### IPL Player Performance Analytics Platform with KPI Dashboard

#### 1. Project Title / Headline

🏏 **IPL Player Performance Analytics Platform with KPI Dashboard**

An interactive Power BI dashboard designed to analyze IPL cricket data, evaluate player and team performance, and generate actionable insights through dynamic KPIs and visual analytics.


#### 2. Short Description / Purpose

The IPL Player Performance Analytics Dashboard is a data-driven Business Intelligence solution developed using Power BI to analyze IPL datasets from 2008–2024. The dashboard enables users to explore player statistics, team performance, venue trends, and season-wise metrics through interactive visualizations and KPI tracking.

This project is intended for sports analysts, cricket enthusiasts, team management, and data analysts seeking insights from historical IPL data. 


#### 3. Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Dashboard development and interactive visualizations.
* 📂 **Power Query** – Data cleaning, transformation, and preprocessing.
* 🧠 **DAX (Data Analysis Expressions)** – KPI calculations and custom measures.
* 🐍 **Python** – Data analysis and preprocessing.
* 🗄️ **SQL** – Data querying and aggregation.
* 📈 **Excel** – Initial data cleaning and validation.
* 🔗 **Data Modeling** – Relationships between IPL match and ball-by-ball datasets.
* 📁 **File Format** – `.pbix` for development and dashboard deployment. 


#### 4. Data Source

**Source:** Kaggle IPL Datasets (2008–2024)

The project utilizes historical IPL datasets containing:

* Match details
* Team information
* Venue information
* Ball-by-ball delivery records
* Batting statistics
* Bowling statistics
* Extras and dismissal information

The data was cleaned, transformed, and modeled to support KPI generation and interactive analytics. 


#### 5. Features / Highlights

### • Business Problem

The IPL generates massive volumes of cricket data every season. Extracting meaningful insights from raw match records can be challenging for analysts, teams, and cricket enthusiasts.

Key questions include:

* Which players consistently perform across seasons?
* Which teams dominate playoff qualifications?
* How do venues influence match outcomes?
* What trends exist in scoring patterns over time?
* Which bowlers and batsmen deliver the highest impact?


### • Goal of the Dashboard

To create an interactive analytics platform that:

* Evaluates player and team performance.
* Tracks season-wise IPL trends.
* Provides venue and match insights.
* Supports data-driven sports analysis.
* Visualizes key performance indicators through intuitive dashboards.
  

### • Walkthrough of Key Visuals

#### Key KPIs

* Total Seasons Analyzed
* Total Teams
* Total Matches
* Strike Rate
* Economy Rate
* Total Runs
* Average Runs
* Wickets Taken


#### IPL Overview Dashboard

Provides a high-level view of IPL statistics including:

* Total matches played
* Team participation
* Season-wise scoring trends
* Wide ball and no-ball analysis
* Top stadiums by match count
* Most frequently used umpires
* Qualifying teams across seasons


#### Player Profile Dashboard

Interactive player analysis including:

* Top Run Scorers
* Top Wicket Takers
* Top Boundary Hitters
* Strike Rate Analysis
* Economy Rate Analysis
* Season-wise player performance


#### Interactive Filters & Slicers

Users can dynamically filter dashboards by:

* Season
* Player
* Team
* Venue
* Performance metrics


### • Business Impact & Insights

#### Performance Evaluation

Helps identify consistent performers such as batsmen and bowlers across multiple IPL seasons.

#### Strategic Decision-Making

Supports team management and analysts in understanding performance patterns and player effectiveness.

#### Trend Analysis

Reveals season-wise scoring growth and changing gameplay dynamics in T20 cricket.

#### Venue Intelligence

Highlights stadium utilization and venue-based performance differences.

#### Sports Analytics

Demonstrates how data analytics and business intelligence techniques can be applied to sports datasets for actionable insights. 


#### 6. Key Insights Obtained

* Average runs per innings have increased across IPL seasons, indicating more aggressive batting strategies.
* Mumbai Indians and Chennai Super Kings have demonstrated strong consistency in playoff qualifications.
* A small number of stadiums host the majority of IPL matches.
* Players such as KL Rahul, David Warner, and Dwayne Bravo have maintained strong performance metrics over multiple seasons.
* Wide balls occur more frequently than no-balls, highlighting common bowling discipline challenges.
